# Cómo observar Plesk sin copiarlo

Para usar Plesk como referencia, conviene documentar flujos y estructura, no copiar diseño, textos ni assets.

## Información útil

- Menú lateral completo.
- Pantallas principales por módulo.
- Flujo para crear dominio.
- Flujo para crear sitio web.
- Flujo para emitir SSL.
- Flujo para crear base de datos.
- Flujo para backups.
- Roles y permisos disponibles.

## Formato recomendado

Por cada pantalla:

- Nombre.
- Propósito.
- Acciones principales.
- Campos.
- Validaciones visibles.
- Estados de error.
- Estados de éxito.

## Evidencia aceptada

- Capturas con datos sensibles ocultos.
- Video corto navegando.
- Lista manual de menús.
- Export de configuración sin secretos.
